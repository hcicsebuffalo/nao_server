School Librarianship MS
The School Librarianship MS program, registered by the New York State Education Department, is authorized to recommend you for certification for the library media specialist, kindergarten through grade 12 initial certificate. This program focuses on preparing you to teach in a 21st century school library. All courses include the latest research-based, best practices occurring in the school library field. You will be exposed to strategies and techniques for teaching information literacy, based on new standards of practice as outlined in the Common Core Learning Standards and the Annual Professional Performance Review.
